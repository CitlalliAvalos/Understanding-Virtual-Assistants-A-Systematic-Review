# Understanding-Virtual-Assistants-A-Systematic-Review
Supplementary dataset of the article "Understanding Virtual Assistants: A Systematic Review". This repository contains the included studies, extracted text fragments, source references, analytical interpretations, and the coding of the reported limitations and challenges.
